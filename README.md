# مطب دندانپزشکی دکتر نادیا روشنی

صفحه فرود نوبت‌گیری برای مطب دندانپزشکی دکتر نادیا روشنی

## راه‌اندازی

```bash
npm install
npm run dev
```

سپس به آدرس [http://localhost:3000](http://localhost:3000) بروید.

## تنظیمات

- **عکس دکتر**: فایل عکس دکتر را با نام `doctor.jpg` در پوشه `public/` قرار دهید
- **شماره تلفن**: در فایل `src/app/page.tsx`، مقدار `href="tel:"` را با شماره تماس مطب به‌روزرسانی کنید (مثال: `href="tel:02112345678"`)

## ساخت برای انتشار

```bash
npm run build
npm start
```
